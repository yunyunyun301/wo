新加入的机制：
1. 麻痹：麻痹状态下的宝可梦速度降低一半，并且有25%的概率无法行动。
2. 睡眠：睡眠状态下的宝可梦无法行动，持续多个回合，每回合有概率苏醒
3. 烧伤：烧伤状态下的宝可梦每回合损失一定的HP，并且攻击力降低一半。
4. 变小：变小状态下的宝可梦闪避率提升，搭配使用电磁波非常猥琐
